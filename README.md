# Project7_Custom-Cursor_Animation-HTML-CSS-Js-GSAP
This project showcases a smooth and interactive custom cursor animation built using HTML, CSS, JavaScript, and GSAP. The default cursor is replaced with an animated cursor that follows the mouse movement with fluid transitions, creating a modern and engaging user experience.
